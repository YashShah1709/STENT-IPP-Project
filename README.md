# STENT IPP Project

## Overview
This repository contains my Individual Project (IPP) on the design and analysis of a stent.  
The project focuses on understanding how stents are used in biomedical engineering to support blood vessels and improve blood flow in patients with cardiovascular diseases.

## Objective
The main objective of this project is to study the structure, function, and importance of stents in medical applications.  
It also explores the engineering concepts involved in stent design and their role in improving healthcare outcomes.

## Key Topics Covered
- Introduction to cardiovascular diseases
- What is a stent and why it is used
- Types of stents used in medical treatment
- Materials used in stent manufacturing
- Design considerations for stents
- Applications and future improvements in stent technology
